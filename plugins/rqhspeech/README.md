# rqhspeech 插件

一个基于 NcatBot 的群聊发言统计管理程序，支持日榜、周榜、月榜、季榜和年榜等多种维度的发言统计与排行。

## ✨ 主要功能

- **多维度排行榜**：支持日榜、周榜、月榜、季榜、年榜及历史榜单查询
- **个人发言统计**：实时查看今日、本周、本月及本年度的发言数据
- **用户管理**：自动注册、自定义用户名、管理员权限控制
- **白名单模式**：支持指定群聊使用或全群开放
- **数据归档**：支持历史数据归档，并导出 JSON 和 CSV 格式文件
- **详细日志**：自动记录每日发言明细，方便追溯与分析

## 🚀 快速开始

### 环境要求

- Python 3.8+
- NcatBot 框架

### 安装步骤

1. 将 `rqhspeech` 文件夹放入 NcatBot 的 `plugins` 目录下
2. 确保已安装依赖库（如有）
3. 重启 NcatBot 加载插件

### 初始配置

插件首次运行会自动生成 `speech_config.json` 配置文件：

```json
{
  "allowed_groups": [],
  "admin_users": [],
  "use_whitelist_mode": false
}
```

- `allowed_groups`: 允许使用插件的群号列表
- `admin_users`: 管理员 QQ 号列表
- `use_whitelist_mode`: 是否开启白名单模式（true=仅白名单群可用，false=所有群可用）

## 📂 目录结构

```
rqhspeech/
├── plugin.py              # 插件主入口
├── speech_config.py       # 配置管理类
├── data_manager.py        # 数据管理模块
├── command_handler.py     # 命令处理器
├── archive_manager.py     # 归档管理器
├── speech_config.json     # 配置文件（自动生成）
├── base_data/             # 数据存储目录
│   ├── users/             # 用户数据文件
│   ├── daily/             # 每日统计数据
│   └── weekly/            # 每周统计数据
├── daily_archives/        # 归档数据
│   ├── rankings/          # 排行榜文件 (JSON/CSV)
│   └── backup_*/          # 自动备份目录
└── logs/                  # 运行日志
```

## ⚙️ 高级配置

### 排行榜人数限制

在 `speech_config.py` 中可调整各榜单显示人数：

```python
self.RANKING_DAILY_TOP_N = 15   # 日榜前15名
self.RANKING_WEEKLY_TOP_N = 10  # 周榜前10名
self.RANKING_MONTHLY_TOP_N = 10 # 月榜前10名
```

### 数据归档策略

- **自动归档**：管理员发送 `自动归档` 命令，系统会自动检查并归档上次备份后的所有日期
- **备份机制**：归档前会自动在 `daily_archives/backup_日期/` 目录下备份用户数据
- **文件格式**：同时生成 `.json`（详细数据）和 `.csv`（表格数据）两种格式

## 🔧 开发说明

### 代码架构优化

本版本根据 got.txt 建议进行了以下优化：
1. **模块化设计**：将命令处理、数据管理、归档逻辑拆分为独立模块
2. **全局单例**：管理器采用全局单例模式，避免重复实例化
3. **类型统一**：统一 `group_id` 和 `user_id` 为字符串类型，防止比较错误
4. **日志系统**：使用标准 logging 模块替代手写日志
