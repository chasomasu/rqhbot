# 示例插件集合

本目录包含各种示例插件，帮助开发者快速了解 RqhBot 插件开发。

## 📁 示例插件列表

### 1. hello_plugin - 基础问候插件
- **功能**: 响应"你好"、"hello"等问候语
- **特点**: 展示基本的消息处理和回复功能
- **适用**: 初学者入门

### 2. admin_plugin - 管理员插件
- **功能**: 提供管理员命令，如禁言、踢人等
- **特点**: 展示权限控制和高级 API 使用
- **适用**: 中级开发者

### 3. game_plugin - 游戏插件
- **功能**: 提供猜拳、骰子等小游戏
- **特点**: 展示娱乐功能和随机数生成
- **适用**: 想添加娱乐功能的开发者

### 4. stats_plugin - 统计插件
- **功能**: 统计群消息数量、活跃用户等
- **特点**: 展示数据持久化和统计分析
- **适用**: 需要数据统计功能的开发者

## 🚀 如何使用示例插件

1. 将示例插件文件夹复制到 `plugins/` 目录
2. 确保插件文件夹中包含 `main.py` 文件
3. 重启机器人或重新加载插件

## 💡 创建自己的插件

参考 [插件开发指南](../docs/06_PLUGIN_DEVELOPMENT.md) 了解如何创建自定义插件。

## 📋 插件开发最佳实践

1. **单一职责**: 每个插件只负责一个明确的功能
2. **错误处理**: 妥善处理异常情况，提供友好提示
3. **配置管理**: 使用配置文件管理插件设置
4. **日志记录**: 适当记录关键操作和错误信息
5. **性能优化**: 避免阻塞操作，合理使用异步
