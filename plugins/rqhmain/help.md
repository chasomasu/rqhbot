# Rqhmain 插件使用指南

## 天气查询

- `天气 + 城市名` - 查询当前天气
- `预报 + 城市名` - 查看天气预报
- `降雨/湿度/风力 + 城市名` - 查看指定天气要素

## 新闻资讯

- `新闻` / `60秒` / `热点` - 获取 60 秒读懂世界

## 运势占卜

- `运势` / `今日运势` - 获取今日运势
- `八字` / `命理` / `紫微` / `星座` / `塔罗` - 运势占卜查询

## 图片生成

- `发图 + 字符序列` - 生成并发送组合图片
