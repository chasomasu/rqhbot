# Rqhshen 修仙系统

这是一个基于QQ机器人的修仙模拟系统，玩家可以通过各种指令进行修仙体验。

系统包含250个修仙境界，从练气期到无上至尊龙，每个境界还细分为9个子境界，总共超过2000个修仙等级。

## 功能说明

- `开灵` - 激活体内灵力，正式踏入仙途（有几率失败）
- `打坐` - 进行修炼，获取随机修为，获得当前修为20%-5000%的经验（最低10点，最高1000万）
- `突破` - 尝试突破到下一境界（需满足修为要求，但有概率失败损失修为）
- `飞升` - 在指定境界尝试飞升（成功率30%，失败减半修为）- 在9、49、159、199、279级可飞升
- `挑战 QQ号` 或 `挑战 昵称` - 挑战其他玩家进行对战
- `攻击 QQ号` 或 `攻击 昵称` - 攻击其他玩家
- `修仙状态` - 查看自己的修仙状态
- `修仙排行` - 查看修仙排行榜
- `修仙帮助` - 查看帮助信息

## 游戏机制

1. **境界系统**：共250个主境界，每个主境界分为9个子境界，总共2259个子境界加上最终境界，合计2260个境界
2. **修为系统**：通过打坐获得修为，修为达到阈值后自动突破
3. **突破系统**：手动突破有概率失败，失败会损失部分修为
4. **飞升系统**：在9、49、159、199、279级主境界可以尝试飞升，成功率30%，失败扣除一半修为
5. **挑战系统**：可与其他玩家进行对战，胜利获得修为，失败损失部分修为
6. **攻击系统**：攻击其他玩家，成功时对方扣一半修为，自己扣1/3修为
7. **排行榜系统**：实时显示修为排行榜

## 飞升说明

飞升是修仙的重要节点，只有在特定境界才能尝试飞升：

- **可飞升境界**：9级、49级、159级、199级、279级
- **飞升成功率**：30%
- **飞升失败惩罚**：扣除一半修为
- **飞升成功效果**：
  - 获得飞升称号
  - 记录飞升次数
  - 继续在当前境界积累修为

飞升成功后，玩家将获得"已飞升"状态，并记录飞升次数。279级飞升后可以继续积累修为，但无法突破到更高境界。
