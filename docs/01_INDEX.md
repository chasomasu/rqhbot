# RqhBot 文档导航

基于 NapCat OneBot11 协议的 Python QQ 机器人 SDK。

## ⚡ 快速开始

```bash
pip install -r requirements.txt
cp config.yaml.example config.yaml   # 编辑 config.yaml 配置连接信息
python run.py                         # 启动机器人
```
